# バズ解析スキル（トレンドWeb調査 → 投稿ネタ案）

福岡県柳川市の MIX BAR「YANAGAWA BANKSY（柳川バンクシー）」の SNS 集客のために、
**今バズっている投稿の傾向を Web で調査し、お店で再現できる具体的な投稿ネタ案に変換する**ためのスキル。

オーナー（非エンジニア）は「効果・売上に効くか」を重視する。技術的な分析よりも、
**「明日この動画を撮ればバズりそう」というレベルの具体案**を出すことがゴール。

---

## 🎯 このスキルのゴール

1. SNS（主に TikTok / Instagram、必要に応じて X・Google）で**今バズっている投稿の傾向**を出典付きで把握する
2. その傾向を**柳川バンクシーの資産（キャスト・料金・曜日特典・立地・コンセプト）にマッピング**する
3. **すぐ撮影・投稿できる具体的なネタ案**を、優先度つきランキングで提示する

## 🔁 作業手順

### STEP 0. スコープを確認
ユーザーに対し、必要なら1問だけ確認する（既に指定があれば飛ばす）：
- どのプラットフォーム中心か？（TikTok / Instagram リール / 両方）
- 何を打ち出したいか？（キャスト / 店内の雰囲気 / 料金 / 曜日特典 / 観光客向け 等）

指定がなければ **TikTok + Instagram リールの両方／キャスト＆雰囲気推し** をデフォルトにして進める。

### STEP 1. Web 調査（複数の角度でファンアウト）
深く・出典付きで調べたいときは **`deep-research` スキル**を使うのが最適。
軽く済ませるなら `WebSearch` / `WebFetch` を直接複数回叩く。**必ず複数の検索角度を使う**：

1. **業態トレンド**：「ガールズバー TikTok バズ 2026」「バー リール 伸びる 企画」「キャバ TikTok 流行」
2. **フォーマット/演出**：「TikTok 飲食店 バズ フォーマット」「ナイト系 ショート動画 構成 フック」流行りの音源・編集・テロップ
3. **地域/観光**：「福岡 夜 TikTok 人気」「柳川 観光 SNS」「川下り グルメ Instagram」
4. **競合・類似店**：福岡・九州のバー/ガールズバー/スナックの伸びてるアカウントの投稿傾向
5. **季節要因**：今の月（実行時の日付を確認）に合うイベント・行事・記念日ネタ

> 注意：日付や「今」のトレンドは記憶に頼らず**必ず Web で裏取り**する。
> ソースは TikTok/Instagram の実投稿、まとめ記事、マーケ系メディアなど複数当たる。

### STEP 2. 出典を確認し、トレンドを抽出
- 各トレンドについて「フォーマット（尺・構成）」「フック（最初の1〜2秒）」「なぜ伸びるのか」を1〜2行で言語化
- 出典 URL をメモ（あとで提示する）
- **再現性で足切り**：器材・予算・人員的にお店で撮れないものは落とす（オーナーは現場で回せる範囲を好む）

### STEP 3. お店の資産にマッピング
抽出した各トレンドを、上の「お店の前提」表のどの資産で再現するか紐づける。
例：「ビフォーアフター系がバズ → 来店前/酔った後のテンション差」「ランキング動画 → キャスト人気投票」など。

### STEP 4. 投稿ネタ案を出力（メイン成果物）
**5〜8案**を優先度順（バズ期待 × 実現しやすさ）で。各案を次のテンプレで：

```
### 案◯：◯◯（キャッチーなタイトル）
- 🎬 フォーマット：尺 / 構成（例：15秒・3カット・テロップ多め）
- 🪝 フック（最初の2秒）：◯◯
- 📝 キャプション案：◯◯（絵文字込み・実際に貼れる文面）
- #ハッシュタグ：#柳川 #柳川バー #ガールズバー …（5〜10個）
- 🎯 狙う客層：◯◯
- 📈 なぜバズる：参考トレンド「◯◯」(出典) に基づき◯◯
- 🛠 実現難易度：★☆☆（誰が・何分で撮れるか）
- 🏪 お店要素：どのキャスト/曜日特典/立地を使うか
```

最後に：
- **今週まず撮るべき1〜2案**を太字で名指し（オーナーが迷わないように）
- **参考にしたトレンドの出典リスト**（URL）
- 撮影時のワンポイント（縦動画・最初の1秒・店名/駅徒歩20秒は必ず入れる 等）

## ✅ ルール・注意点
- **出典必須**。「バズってる」と言うなら根拠の URL を添える。憶測だけで断定しない。
- **誇大・違法表現 NG**：景表法/薬機法的に危ない言い回し（「絶対」「日本一」等）や、
  料金の誤認を招く表現は避ける。料金は税抜・別途TAX10%である点を崩さない。
- **キャストのプライバシー配慮**：本人が出る前提のネタは「出演OKな範囲で」と明記する。
- **再現可能なネタ限定**：スマホ1台＋店内で撮れる案を優先。大掛かりな機材前提の案は出さない。
- トレンドは水物。**実行時点の日付（環境のcurrentDate）を確認**してから「今」を語る。
- 仕上げは日本語・カジュアル絵文字OK。技術用語より「効果・売上」に寄せて説明する。
